## Direction

The mascot should feel like the same character in two moments:

- Thoughtful/bored-but-hopeful at the desk
- Energetic/confident with the rocket pack

The flying mascot should feel like the desk mascot rotated into action, not a second mascot design.

## Palette

- Paper: `#FAF7F1`
- Navy: `#003851`
- Mint: `#59CB89`
- Amber: `#EF8F26`
- Violet: `#7C3AED`, only as a tiny sparkle

## Wordmark

Use `JobBored` only. The final `d` should stay clean and normal. No extra tail or flourish after the final letter.
