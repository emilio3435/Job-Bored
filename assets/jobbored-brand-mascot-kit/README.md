## Core Rules

- Keep one mascot model: same oval body, same eyes, same antenna, same line weight.
- Use the rocket-pack pose for energy/progress moments.
- Use the thought-bubble/laptop pose for thinking, setup, empty states, and guidance.
- Use the pose sheet for UI spot illustrations only; do not treat those as new logo variants.
- Wordmark must spell `JobBored` exactly.
- The final `d` in `Bored` stays clean: no extra swash, tail, flourish, or detached mark.

## Palette

## Notes

These are high-quality raster/source exports from GPT image generation plus local crops/composites. For production-critical vector use, trace the chosen lockups from the source sheets instead of generating more unrelated mascot variants.
