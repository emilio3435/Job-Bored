⚡ Boost job match quality (recommended) Checking…

Your worker uses Google Jobs (via SerpApi) as its highest-quality source of real, live job openings. Google has already indexed every job posted through Greenhouse, Lever, Ashby, Workday, and 100+ other ATS platforms — so this one integration replaces the need for brittle career-page scraping. Without it, the worker still runs, but you'll get far fewer matches.

  1. Create a free account at serpapi.com — the free tier includes 100 searches/month (~20 daily discovery runs).
  2. Copy your API key from the SerpApi dashboard.
  3. Open integrations/browser-use-discovery/.env and add a new line: SERPAPI_API_KEY=paste-your-key-here
  4. Restart your local worker (npm run discovery:worker:start-local) so the new env var is picked up, then run discovery again.
