Connect Google

Get a Client ID

Free, ~3 minutes. JobBored is BYO keys — your Client ID stays in your browser. We’ve copied this site’s origin to your clipboard so you can paste it when Google asks.

Step-by-step guide (open if it’s your first time)
  1. In Google Cloud Console, create or pick a project (top bar → project picker → New project).
  2. Configure the OAuth consent screen (left nav → APIs & Services → OAuth consent screen). Pick External, fill in the app name + your email, save. You can add yourself as a test user under Audience to skip “unverified app” warnings.
  3. Enable the Google Sheets API and Google Drive API for the project (APIs & Services → Library → search → Enable).
  4. Open Credentials → Create credentials → OAuth client ID → application type Web application.
  5. Under Authorized JavaScript origins, click Add URI and paste the origin shown above (already on your clipboard). Leave redirect URIs empty — JobBored doesn’t use them.
  6. Click Create. Google shows a popup with your Client ID — copy it and paste below.

Stuck? The Client ID always ends in .apps.googleusercontent.com. If you see a “redirect_uri_mismatch” later, you picked the wrong app type — recreate as Web application.
